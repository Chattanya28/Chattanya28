## 🔐 Featured Project: Sentinel Core

**A SIEM-style security dashboard I built to understand how threat detection and ML actually work.**

I was completely new to SIEM tools and Machine Learning. So instead of watching tutorials forever, I built Sentinel Core to figure out:
- How security teams triage real-time alerts
- What "anomaly confidence thresholds" actually mean
- Why context matters more than raw logs

It has a live threat map, real-time event feeds, ML-based risk scoring, and automated playbooks. Is it production-ready? No. Did I learn more than any course could teach me? Absolutely.
